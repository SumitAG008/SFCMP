## Where to Enter Company ID and User ID

The input fields are located in the **"Compensation Plan Information"** panel at the top of the page:

1. **Company ID** - Should already be pre-filled with "SFHUB003674"
2. **User ID** - Enter your SuccessFactors User ID here
3. **Form ID** - (Optional) Enter the Form ID if you have one
4. **Effective Date** - (Optional) Select the effective date

## How to Use the Application

### Step 1: Enter Required Information
1. Look for the **"Compensation Plan Information"** panel
2. **Company ID** should already show "SFHUB003674"
3. **Enter your User ID** in the "User ID" field
4. (Optional) Enter Form ID and Effective Date

### Step 2: Load Data
1. Click the **Refresh** button (circular arrow icon) in the header
2. This will fetch compensation data from SuccessFactors
3. The data will appear in the "Compensation Worksheet" table below

### Step 3: Edit Compensation Data
1. Once data is loaded, you can edit:
   - Merit Increase (% or Amount)
   - Adjustment Increase (% or Amount)
   - Lump Sum
   - Final Salary
   - Status
   - Comments

2. Calculations happen automatically:
   - When you enter percentages, amounts are calculated
   - When you enter amounts, percentages are calculated
   - Total Pay is calculated automatically

### Step 4: Save Changes
1. After making edits, click the **Save** button (floppy disk icon)
2. This will send the updated data to SuccessFactors
3. You'll see a success message when saved

## Understanding the Warning

The warning "Please enter Company ID and User ID" appears when:
- You click **Refresh** or **Save** buttons
- But haven't entered the required Company ID and User ID

**Solution**: Just enter your User ID in the "User ID" field (Company ID is already filled), then click Refresh.

## If You Don't See the Input Fields

If the "Compensation Plan Information" panel is not visible:

1. **Scroll up** - The panel is at the top of the page
2. **Check if panel is collapsed** - Click to expand it
3. **Hard refresh** - Press `Ctrl+Shift+R` to reload

## Troubleshooting

### Warning keeps appearing
- Make sure you've entered both Company ID and User ID
- Company ID should be "SFHUB003674" (pre-filled)
- User ID must be your actual SuccessFactors user ID

### No data after Refresh
- Check browser console (F12) for errors
- Verify your User ID is correct
- Check if SuccessFactors API is accessible

### Can't see input fields
- Scroll to top of page
- Look for "Compensation Plan Information" panel
- Try hard refresh (Ctrl+Shift+R)
